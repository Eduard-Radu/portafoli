AparCar

AparCar és una pàgina web para el concurso HackEPS Lleida del año 2024, en la cual puedes crear una cuenta, entrar con ella, buscar parking y ver sus plaças disponibles, a parte de poder reservar una plaça para los siguientes dias y ver la disponibilidad de la semana basandose en un algoritmo que calcula las plaças que se estima que estarán disponibles segun las semanas anteriores.

Tecnologias usadas:

  • Vue.js
  • Javascript
  • HTML
  • CSS
  • Google Firebase (database)
